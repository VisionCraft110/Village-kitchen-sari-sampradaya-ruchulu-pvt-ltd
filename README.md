# 🍽️ Village Kitchen Website

A beautiful and responsive restaurant website for **Village Kitchen**, showcasing traditional village cuisine, menu, specials, and franchise opportunities.

---

## 🌟 Features

- 🎥 Hero section with image slider
- 🍛 Special dishes & today's highlights
- 📋 Menu preview section
- 🖼️ Gallery for franchise & food images
- 📱 Fully responsive (mobile-friendly)
- 📲 WhatsApp Franchise Booking Integration
- 🍃 Traditional themed UI (village style design)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom styling + responsive design)
- JavaScript (Vanilla JS)
- Google Fonts (Playfair Display, Poppins)
